### BDD
* 클라이언트는 구매 금액을 입력한다.
* 구매 금액에 해당하는 로또를 발권해준다.
    * 랜덤한 로또 번호 생성
    * 루프를 돌면서 랜덤으로 생성된 번호를 기반으로 로또 각각의  발급
* 지난 주 당첨번호를 입력한다.
* 클라이언트가 구매한 로또가 맞는지 확인한다.
    * enum class를 통한 매칭 값 확인
    * 통계 로직을 통한 결과값 표출
